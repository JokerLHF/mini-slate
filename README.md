# mini-slate
使用 ts 实现 slate 富文本。目前实现 slate/example 里的
- richText
- check-list
- embeds
- hover-toolbar
- mentions
- tables
- images

# 开发
pnpm run start

# 测试用例
目前只完成 slate-core 一部分测试用例，进入 slate 启动 pnpm run test
